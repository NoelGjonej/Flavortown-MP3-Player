# MP3 Player

This is a Spotify-Like website that alows you to listen to pre-added music, create playlists and create queues.<br>

# How to use

You need to download the zip folder, unzip it and open it with VS Code.
Then you need to download a library called "Live Server".
Then you need to open the main.html file and right-click on it.
A window will pop up and then you need to press open with Live Server.
Now you'll be able to use the website.
You can create playlists, add songs to playlist by using the "+" button.
And you can add songs to the queue by right-clicking on them.
